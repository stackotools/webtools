Frequently Asked Questions

What makes a password strong?

A strong password is long (12+ characters), uses a mix of character types (upper/lower case, numbers, symbols), and doesn't contain personal information or dictionary words.

How are the passwords generated?

Passwords are generated using cryptographically secure random number generation in your browser. No data is sent to our servers.

Should I save my passwords here?

No, the password history is stored only in your browser temporarily. For permanent storage, use a dedicated password manager.

Why is my password strength low?

Password strength depends on length and complexity. Try increasing length or adding more character types to improve strength.
